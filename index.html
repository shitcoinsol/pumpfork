Developer Quickstart

Install CLI
curl -sSL https://get.pumpfork.org | bash
# then
pf --version
Create a project
pf init my-app
cd my-app
pf build
pf deploy --network testnet
Connect (TypeScript)
import { Connection, Keypair } from "@pumpfork/web3";

const connection = new Connection("https://rpc.testnet.pumpfork.org");
const wallet = Keypair.generate();

(async () => {
  const height = await connection.getSlot();
  console.log("slot:", height);
})();

Pumpfork Documentation (testnet)

Pumpfork is a pragmatic fork of Solana focused on high‑throughput execution and meme‑native governance.

Tip: If you’ve built on Solana, you already know 90% of Pumpfork. Tools and SDK names mirror Solana equivalents.

Write & Deploy a Program

Programs are eBPF-compatible and built with the Pumpfork SDK. Below is a minimal TypeScript client to fetch the current slot.

import { Connection } from "@pumpfork/web3";
const c = new Connection("https://rpc.testnet.pumpfork.org");
const slot = await c.getSlot();
console.log("slot", slot);
